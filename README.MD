# Gaming Backlog Tracker

A streamlined desktop application primarily aimed at **PlayStation trophy hunters**, designed to help gamers organize, track, and manage their collection across multiple platforms. This project demonstrates CRUD (Create, Read, Update, Delete) operations and efficient data management.

## 🚀 Features

*   **Game Management:** Easily add new titles to your personal backlog.
*   **Platform Management:** Add new platforms and categorize your library by gaming system (e.g., PC, PlayStation, Xbox, Switch).
*   **Progress Monitoring:** Track your journey toward 100% completion with a dedicated percentage tracker, perfect for **trophy hunting goals**.
*   **Clean UI:** Simple, intuitive interface built for quick data entry and overview.

## 📋 How It Works

1.  **Add a Game:** Enter the title of the game you've recently acquired.
2.  **Manage Platforms:** Select an existing system (like PlayStation) or add a new platform type to keep your library organized.
3.  **Update Progress:** Track your progress to 100% completion, managing **all trophies** including those from base games and DLC expansions.
